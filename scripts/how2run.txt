2- WindowsKey ->type PowerShell -> rightClik -> open as Administrator

3- go to scripts folder -> In Powershell type -> cd C:\Users\YOURNAME\Downloads\Win10-Purge-master\scripts

     -->   ls -Recurse *.ps1 | Unblock-File

     -->   ls -Recurse *.psm1 | Unblock-File 

4- now run the scripts with " ./ "  example: ./defaultApps
